2. FAQ Block
3. CTA Block

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lsx-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
