LSX Blocks adds adds new blocks to the Gutenberg editor.
This plugins is based on Atomic Blocks https://atomicblocks.com

= Blocks List =

1. Accordion Block
2. Author Profile Block
3. Banner Block
4. Button Blocks
5. Card Block
6. Container Block
7. CTA Block
8. Drop Cap Block
9. Notice Block
10. Post Carousel Block
11. Post Grid Block
12. Sharing Block
13. Spacer Block

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lsx-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
